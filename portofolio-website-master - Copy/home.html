Cyber security

What can I do?

Sql ejection

teknik serangan siber di mana peretas menyisipkan atau "menyuntikkan" perintah SQL (Structured Query Language) berbahaya ke dalam input data yang dikirim ke sebuah aplikasi web. Tujuannya adalah untuk memanipulasi database di belakang aplikasi tersebut.

(DoS/DDoS) Attack

Membanjiri server atau jaringan dengan lalu lintas data yang sangat besar sehingga sistem menjadi lumpuh dan tidak bisa diakses oleh pengguna lain.

Cross-Site Request Forgery (CSRF)

Teknik untuk menipu browser korban agar melakukan tindakan yang tidak mereka inginkan di situs web tempat mereka sudah login. Contohnya, tanpa sadar mentransfer uang atau mengubah kata sandi.
